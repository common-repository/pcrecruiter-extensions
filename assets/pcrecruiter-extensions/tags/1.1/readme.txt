This plugin generates iframe embed code for the PCRecruiter Web Extensions via a simple shortcode.


== Installation ==

1.  Click 'Add New' from the 'Plugins' menu in your WordPress admin panel.
2.  Use the 'Upload Plugin' option and browse to the PCRecruiter-Extensions.zip file.
3.  Activate the plugin.
4.  Edit the page where you want to display the PCRecruiter content and paste in the following:  
      
    [PCRecruiter link=""]  
      
    Place the URL provided to you by Main Sequence between the quotes. If the URL does not begin with 'http', the plugin will assume you intend to load a URL from the PCRecruiter ASP hosting system and will prepend the correct URL.

Optional parameters:

*   initialheight="" (defaults to 640 pixels if omitted)
*   background="" (defaults to transparent if omitted)
